# ROMs Directory

Place your Dragon Warrior IV ROM in this directory.

## Required ROM

**File**: `Dragon Warrior IV (1992-10)(Enix)(US).nes`

### ROM Information

| Property | Value |
|----------|-------|
| Name | Dragon Warrior IV |
| Region | USA |
| Release | October 1992 |
| Publisher | Enix |
| System | Nintendo Entertainment System (NES) |
| Mapper | MMC3 (Mapper 4) |
| PRG ROM | 256 KB (16 × 16 KB banks) |
| CHR ROM | 128 KB (16 × 8 KB banks) |
| Mirroring | Vertical |

## Usage

The build system expects the ROM at:
```
.\roms\Dragon Warrior IV (1992-10)(Enix)(US).nes
```

If you have a differently named ROM, you can:
1. Rename it to match the expected name
2. Update the path in `build.config.json`
