## Map Data RAM Address

Current map data is stored at `$7800` in WRAM.

### Tile Format
Each byte: `RRRttttt`
- `ttttt` (bits 0-4): Tile number 0-31
- `RRR` (bits 5-7): Roof number 0-7 (0 = no roof)

### Related RAM Addresses
| Address | Size | Description |
|---------|------|-------------|
| $063 | byte | Current Map Number |
| $064 | byte | Current Sub-Map Number |
| $6F20 | $20 | Smoothing Flags |
| $6F40 | $40 | Tile Behaviors |
| $7600 | $C0 | Tileset Numbers |
| $76C0 | $40 | Tile Attributes |
| $7800 | varies | Map Data |
